# 08 — Glossary (Thuật ngữ)

| Thuật ngữ | Giải thích |
|-----------|-----------|
| **Qbank (Question Bank)** | Ngân hàng câu hỏi luyện tập |
| **Vignette** | Đoạn mô tả tình huống lâm sàng làm đề bài câu hỏi |
| **Stem** | Phần thân đề bài (vignette + dữ kiện) |
| **Lead-in** | Câu hỏi dẫn ("Chẩn đoán phù hợp nhất là?") |
| **Single Best Answer** | Câu chọn 1 đáp án đúng nhất trong nhiều lựa chọn |
| **Study Mode** | Chế độ học: xem giải thích ngay sau mỗi câu |
| **Exam Mode** | Chế độ thi: giới hạn thời gian, xem kết quả sau khi nộp |
| **Session** | Một phiên làm gồm nhiều câu hỏi |
| **Attempt** | Một lần trả lời một câu trong session |
| **Omitted / Skip** | Câu bỏ qua không trả lời |
| **Flag / Mark** | Đánh dấu câu để xem lại |
| **Mastery** | Mức độ thành thạo theo chủ đề (0–5) |
| **Weak Topic** | Chủ đề yếu (correct rate/mastery thấp) |
| **Heatmap** | Bản đồ nhiệt thể hiện mức độ theo chủ đề/thời gian |
| **Spaced Repetition (SRS)** | Thuật toán lặp lại ngắt quãng cho flashcard |
| **Ease Factor / Interval** | Tham số SRS (độ dễ, khoảng cách ôn) |
| **High-Yield** | Nội dung trọng tâm, hay ra thi |
| **Cross-link** | Liên kết chéo giữa câu hỏi ↔ bài viết ↔ thuốc ↔ ảnh |
| **Library Article** | Bài viết y khoa trong thư viện |
| **Study Plan** | Lộ trình học cá nhân hóa theo ngày thi |
| **Entitlement** | Quyền lợi mở khóa theo subscription |
| **Paywall** | Rào nội dung yêu cầu trả phí |
| **Gating** | Cơ chế chặn/mở tính năng theo quyền/subscription |
| **Percentile** | Thứ hạng phần trăm so với nhóm |
| **Cohort** | Nhóm người dùng cùng đặc điểm (cùng kỳ thi, org) |
| **RBAC** | Role-Based Access Control |
| **IDOR** | Insecure Direct Object Reference |
| **Optimistic Update** | Cập nhật UI trước, đồng bộ server sau |
| **Rollup** | Tổng hợp dữ liệu chi tiết thành số liệu tóm tắt |
| **Feature Flag** | Cờ bật/tắt tính năng |
| **Signed URL** | URL có chữ ký, thời hạn để truy cập media bảo mật |
| **Reverb** | Server WebSocket của Laravel (realtime) |
| **Meilisearch** | Công cụ full-text search |
